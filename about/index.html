About（極小キロバイト文庫）

⏰2026-04-17 👤4番目の人

こんにちは。4番目の人（@4thlog）です。

ここは「極小キロバイト文庫」。 できるだけ小さな構成で、静かに文章を置いていく場所です。

サイトは GitHub Pages と Hugo で作っています。 サーバーに依存せず、広告もなく、余計な機能も持たない。 必要最小限の仕組みで動いています。

デザインは、かつて WordPress で使っていた自作テーマをもとにしました。 装飾よりも、壊れにくさと読みやすさを優先しています。

現在は働きながら資格の勉強をしています。 ノートは Obsidian で管理し、そのマークダウンをそのままここに流しています。 余計な変換をせず、書いた形に近いまま公開しています。

スマートフォンは持っていません。 使っているのは Punkt MP02 と iPad mini だけです。

MP02は社会的な連絡手段。 iPad miniは執筆と調べもの、そして生活に必要な最低限のアプリのために使っています。

大きなことはできません。 けれど、小さく、軽く、長く続く場所でありたいと思っています。

軽いことは、自由であること。
